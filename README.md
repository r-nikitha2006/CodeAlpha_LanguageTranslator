# CodeAlpha_LanguageTranslator

This project was developed as part of the **CodeAlpha Artificial Intelligence Internship**.

## Project Overview
The Language Translation Tool is an AI-based application that translates text from one language to another. It uses a translation API to process the user input and display the translated output.

The application provides a simple graphical user interface where users can enter text and get the translated result instantly.

## Features
- Translate text between languages
- Simple user-friendly interface
- Fast and accurate translations
- Built using Python

## Technologies Used
- Python
- Tkinter (for GUI)
- Googletrans API (for translation)

## How It Works
1. The user enters text into the input field.
2. The program sends the text to the translation API.
3. The API processes the request and translates the text.
4. The translated text is displayed on the screen.
